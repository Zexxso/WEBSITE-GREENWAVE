# GreenWave

!
**"Gelombang perubahan dimulai dari satu tindakan kecil. Bergabunglah bersama GreenWave untuk menciptakan masa depan yang lebih hijau."**

GreenWave adalah platform kampanye digital yang didedikasikan untuk meningkatkan kesadaran, mengedukasi masyarakat, dan mendorong aksi nyata dalam isu-isu lingkungan. Kami percaya bahwa setiap individu memiliki peran penting dalam menjaga keberlanjutan bumi kita.

## âœ¨ Fitur Utama

* **Edukasi Interaktif:** Artikel, infografis, dan video mendalam tentang perubahan iklim, daur ulang, konservasi energi, dan keanekaragaman hayati.
* **Aksi Nyata:** Direktori kegiatan sukarela, *challenge* lingkungan, dan panduan untuk praktik gaya hidup ramah lingkungan.
* **Komunitas:** Forum dan fitur berbagi cerita untuk menghubungkan para pegiat lingkungan di seluruh Indonesia.
* **Donasi & Kemitraan:** Informasi mengenai organisasi nirlaba yang bisa didukung dan peluang kolaborasi.

## ğŸš€ Instalasi & Setup Lokal

Ikuti langkah-langkah sederhana ini untuk menjalankan GreenWave di lingkungan lokal kamu.

### Prasyarat

Pastikan kamu sudah menginstal Git di komputermu.

### Langkah-Langkah

1.  **Clone Repositori:**
    ```bash
    git clone [https://github.com/Zexxxso/WEBSITE-GREENWAVE.git](https://github.com/Zexxxso/WEBSITE-GREENWAVE.git)
    ```
2.  **Masuk ke Direktori Proyek:**
    ```bash
    cd WEBSITE-GREENWAVE
    ```
3.  **Buka di Browser:**
    Karena ini adalah proyek *front-end* statis, cukup buka file `index.html` langsung di *browser* favorit kamu.

*(Tips: Untuk pengembangan yang lebih baik, disarankan menggunakan ekstensi Live Server pada VS Code).*
